# Robotics Sim2Real Lab

This project provides tutorials and tools for reinforcement learning and imitation learning with ROBOTIS robots, enabling the transfer of trained policies from simulation to real hardware (Sim2Real). It serves as a practical framework for experimenting with and deploying learned robotic behaviors.

## Key Features
*   Tutorials for Reinforcement Learning (RL) and Imitation Learning (IL) using ROBOTIS platforms.
*   Sim2Real functionality to deploy trained models on physical robots.
*   Based on the original ROBOTIS-GIT/robotis_lab repository.

## Tech Stack
*   Python
*   ROS (Robot Operating System)
*   OpenAI Gym/Gymnasium
*   PyBullet (for simulation)

## Getting Started
1.  Clone the repository: `git clone https://github.com/zoreanuj/robotics-sim2real-lab.git`
2.  Install dependencies: `pip install -r requirements.txt`
3.  Follow the tutorial scripts in the `examples/` directory to begin training.
